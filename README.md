## Featured Projects

### 🔍 [RecipaAI — Multi-Agent RAG Assistant](https://github.com/faresalnamla) · *Dec 2025*
Designed and developed a multi-agent RAG-based assistant from scratch, handling system architecture, agent coordination, and retrieval workflows. Built Python pipelines for data ingestion, vector indexing, and semantic search using modern LLM frameworks to generate accurate, context-aware responses from unstructured data.

### 📰 [Palestine Fake News Detector](https://github.com/faresalnamla) · *Sep 2025*
Built an NLP-based fake news detection system for Palestinian-related news. Implemented a full pipeline covering text cleaning, tokenization, and TF-IDF feature extraction. Trained and evaluated Decision Tree and Logistic Regression models using accuracy and F1-score as primary metrics.

### 📧 [Email Spam Detection System](https://github.com/faresalnamla) · *Dec 2025*
Developed a spam classifier using Python and TF-IDF for feature extraction. Trained Naive Bayes and Logistic Regression models with a focus on preprocessing quality, feature tuning, and performance evaluation using accuracy and F1-score.
